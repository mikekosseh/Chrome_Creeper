Chrome_Creeper
==============

Autofill enumeration tool for Chrome
Simple C++ app that eumerates the data held in the Web Data sqlite db used by chrome to store all autocomplete data.

Usage

c:\ChromeCreeper.exe
Select option 1 to connect to the DB

Select option 7 to quit the application.
